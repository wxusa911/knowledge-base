# Polymarket 流动性奖励 (Liquidity Farming)

> 通过挂单赚取奖励

---

## 🎯 核心发现

### 如何赚钱

| 方式 | 说明 |
|------|------|
| **挂Maker单** | 在订单簿放置 YES/NO 订单 |
| **提供流动性** | 市场给你发奖励 |
| **案例** | $2000+ USDC 奖励池 |

### 原理

```
1. 在订单簿挂单 (不是立即成交)
2. 等待价格波动成交
3. 获得:
   - 交易手续费 (maker fee)
   - 流动性奖励 (liquidity rewards)
```

## ⚡ 具体操作

### 1. 找到高奖励市场
- 搜索有 **$2000+** 奖励池的市场
- 订单簿薄弱的市场 (容易成交)

### 2. 挂单策略
- 挂 **Maker单** (不是 Taker)
- 放置在买卖价差之间
- 设置合理价格

### 3. 管理
- 像鹰一样盯着订单簿
- 价格不利时 **立即取消**

## ⚠️ 风险

- 订单可能永远不成交
- 价格不利需要手动取消
- 需要实时监控
