### 策略要点

1. **数据源**: NOAA/NWS 官方预报 (免费)
2. **模型**: GFS Ensemble (全球预报系统)
3. **核心**: 找 **市场定价 < 官方预报** 的机会
4. **边缘阈值**: >10% 才下注

## 交易步骤

```
1. 获取 NOAA 72小时预报
2. 对比 Polymarket 市场定价
3. 边缘 >10% → 买入
4. Kelly 仓位管理
```
