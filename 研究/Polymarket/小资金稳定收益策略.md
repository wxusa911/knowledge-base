## ✅ 真正能盈利的策略

### 1. BTC 15分钟短周期套利 (已验证可行)

**项目参考:** https://github.com/sdohuajia/polymarket-bot

| 项目 | 说明 |
|------|------|
| 原理 | 价差套利 + 实时价格监控 |
| 市场 | BTC 15分钟涨跌市场 |
| 触发条件 | 剩余120秒内，价差≥30，概率80%-92% |
| 核心 | WebSocket 实时获取价格，自动下单/止损 |
| 收益 | 根据策略配置 |
| 门槛 | 需要 API Key + 自动化脚本 |

**配置要点:**
```env
TRADE_AMOUNT=5           # 每次下单金额
CONDITION_1_TIME=120     # 剩余120秒
CONDITION_1_DIFF=30      # 价差≥30
C1_MIN_PROB=0.80        # UP概率80%-92%
C1_MAX_PROB=0.92
STOP_LOSS_PROB_PCT=0.15 # 止损线
AUTO_REDEEM=true        # 自动兑奖
```

**风控:**
- 止损线 (STOP_LOSS_DIFF)
- 止盈 (TAKE_PROFIT_RR)
- 每市场最多尝试2次

### 2. 信息差策略 (唯一有效)

```
核心: 你知道市场不知道的事
     = 真正的概率 > 市场定价
```

**案例:**
- 天气交易员: 1,420次预测, 盈利$25,700
- 专业分析师: 65%+胜率

**怎么做:**
1. 获取信息优势 (Twitter/新闻/专业数据)
2. 计算真实概率 vs 市场定价
3. 只买边缘 > 10% 的
