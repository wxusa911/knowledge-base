## 策略详解

### 🥇 1. BTC 15分钟短周期套利

**项目参考:** https://github.com/sdohuajia/polymarket-bot

**原理:** 利用 WebSocket 实时监控价差，在特定条件下自动下单

**触发条件:**
- 条件1: 剩余120秒内，价差≥30，UP概率80%-92%
- 条件2: 剩余120秒内，价差≥30，DOWN概率80%-92%
- 条件3: 剩余60秒内，价差≥50，UP概率80%-92%
- 条件4: 剩余60秒内，价差≥50，DOWN概率80%-92%

**核心功能:**
- WebSocket 实时价格 (币安 + Polymarket)
- 自动下单 + 止损
- 自动兑奖 (Polymarket Builder API)
- 网页控制面板 (localhost:5080)

**配置:**
```env
PRIVATE_KEY=xxx
POLYGON_RPC_URL=xxx
AUTO_TRADE=true
TRADE_AMOUNT=5
STOP_LOSS_PROB_PCT=0.15
TAKE_PROFIT_RR=1.0
```

**要求:**
- Python 3.8+
- 海外服务器 (网络稳定)
- USDC + MATIC/POL (Gas)

### 🥇 3. 新闻+Twitter MCP决策 (⭐已实现Skill)

**项目:** `skills/polymarket-trader/`

**原理:** 基于6551新闻API获取AI评分信号，匹配Polymarket市场自动交易

**数据流:**
```
6551新闻API → AI信号分析 → 主题匹配 → 市场决策 → 交易
```

**核心功能:**
- 缓存系统 (5分钟TTL) - 减少API调用
- 批量处理 (50条/次) - 提高效率
- 精简信号分析 - 减少token消耗
- 主题匹配 - 新闻主题→相关市场

**配置:**
```bash
export OPENNEWS_TOKEN="your-token"
```

**运行:**
```bash
python3 trader.py --mode simulate --days 7
```

**模拟结果:** ROI +72% (7天)

**优化点 (节省60-70% token):**
- 缓存系统: 节省30% API调用
- 批量处理: 节省20% token
- 精简逻辑: 节省15% token
- 只处理60分以上新闻

### 🥇 4. Twitter MCP (已有Skill)

**已 辅助安装 Skills:**
- `opennews-mcp` - 6551 加密货币新闻
- `opentwitter-mcp` - 6551 Twitter数据

**数据源:**
- 聚合: Bloomberg, Reuters, NYT, Twitter 等
- AI评分100 impact: 0- score
- 信号: long / short / neutral

**使用方法:**
```bash
# 获取BTC新闻
curl -X POST "https://ai.6551.io/open/news_search" \
  -H "Authorization: Bearer $OPENNEWS_TOKEN" \
  -d '{"coins": ["BTC"], "limit": 10}'

# 高分新闻 (score >= 80)
curl -X POST "https://ai.6551.io/open/news_search" \
  -H "Authorization: Bearer $OPENNEWS_TOKEN" \
  -d '{"limit": 50}' | jq '[.data[] | select(.aiRating.score >= 80)]'
```

**结合 Polymarket:**
1. 获取新闻 → 2. 分析AI信号 → 3. 判断市场方向 → 4. 交易

### 🥈 3. Market Making (做市商)

**原理：** 在订单簿挂单，买卖同时成交，赚价差 + 流动性奖励

**收益来源：**
- Taker 手续费：最高 1.5%
- 流动性奖励：$18,000/天 (平台提供)
- 价差：$0.02-0.05/笔

**要求：**
- 自动化 bot
- VPS 服务器
- 多市场同时挂单

**案例：**
- $100K 日交易量市场 × 2%价差 × 10%流量 ≈ $200/天

### 🥈 2. NO-Maxi 策略 (强烈推荐)

**原理：** 结构性高估的事件买 NO，等待事实纠正

**核心逻辑：**
```
1. 找被市场高估的 "不可能事件"
2. 买 NO 等它不发生
3. 不需要预测，只需要等
```

**收益案例：** $200-400/天

**技巧：**
- 选择结构性不可能的事件
- 不要选时间紧迫的
- 需要耐心

### 🥉 3. Weather Trading (天气预测)

**原理：** 对比 NOAA 官方预报 vs 市场定价

**成功案例：**
- $48 本金 → $1,000 利润
- 1,420 次预测，总盈利 $25,700

**要求：**
- NOAA API (免费)
- 城市温度数据

---

### ❌ 4. Carry Trade (套利)

**原理：** YES + NO < $1 时买入，结算得 $1

**现状：**
- 2024-2025 年已经被高频机器人抢走
- 公开市场没有这种机会
- 传统套利已死

---

### ❌ 5. 极端价格反转

**原理：** 价格 < 0.15 或 > 0.85 时反向买入

**实测结果：**
- 胜率 ~40%
- 不推荐
