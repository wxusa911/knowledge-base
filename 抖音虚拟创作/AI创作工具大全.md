## 📹 AI 漫剧制作流程 (6步)

1. **脚本/素材库** - 用 ChatGPT 搭建
2. **分镜设计** - AI 生成关键帧
3. **图生视频** - Runway/Pika/即梦
4. **嘴型同步** - Wav2Lip
5. **配音/音效** - 11Labs + 剪映
6. **剪辑/字幕** - 剪映

## 🎯 魔食趣项目 (品牌IP)

### 品牌信息

| 项目 | 内容 |
|------|------|
| 品牌名 | 魔食趣 |
| 定位 | 食品 + 魔法 + 有趣 |
| 色调 | 红色 (#FF3333) |

### IP 形象: 魔食趣小火人

| 项目 | 内容 |
|------|------|
| 名称 | 魔食趣小火人 / 贪吃小魔怪 |
| 性格 | 贪吃、好奇、呆萌、调皮 |
| 特长 | 发现美食、创造美食魔法 |

### 形象关键词

```
可爱红色小精灵、大眼睛带高光、圆润胖乎乎身体、
短短四肢、粉色脸颊、小恶魔尖角、
拿着筷子吃面条、周围金色星星环绕
```

### 应用场景

| 场景 | 用途 |
|------|------|
| 抖音头像 | 简化版精灵形象 |
| 短视频 | 动态表情包 |
| 直播间 | 虚拟主播互动 |
| 品牌周边 | 贴纸、公仔 |

### 制作工作流

1. **形象生成** - Midjourney/即梦AI 生成小火人形象
2. **表情包制作** - 不同表情变化
3. **视频制作** - 图生视频 (Runway/Pika)
4. **直播** - 数字人形象接入
