# 牛景胜抖音账号重启方案

## 第一性原理分析

### 他是谁？

牛景胜不是一个"抖音博主"，而是一个**有432次救援经验的普通人**。

这个身份背后是什么？

| 标签 | 背后的人性需求 |
|------|----------------|
| 432次救援 | **真實** → 观众厌恶假货 |
| 负债做好事 | **衝突** → 故事张力 |
| 蓝天救援队 | **權威** → 专业背书 |
| 河南农民 | **接地气** → 真实感 |
| 太极拳师 | **差异化** → 不做作 |

---

### 核心矛盾

他身上有一个**顶级故事结构**：

```
一个负债的人，自费去做救援，
432次出生入死，不收钱，
还要被误解、被质疑...
```

这是什么？**这是大片剧本都不敢这么写的故事。**

但他自己没意识到这个问题。

---

### 抖音的底层逻辑

抖音不是"展示平台"，而是**注意力拍卖所**。

| 维度 | 真相 |
|------|------|
| 算法 | 争夺用户注意力 |
| 推送 | 前3秒定生死 |
| 留存 | 完播率才是王 |
| 推荐 | 赛马机制 |

**用户为什么刷抖音？**
- 消磨时间 → 需要爽感
- 获取信息 → 需要价值
- 情感共鸣 → 需要代入

**牛景胜能提供什么？**
- 爽感：救援成功、危机解除
- 价值：急救技能、生存知识
- 情感：逆境正能量、真实故事

---

### 为什么以前做不起来？

我推测：

| 问题 | 原因 |
|------|------|
| 没人看 | 内容自嗨，不懂算法 |
| 不持续 | 当副业，没系统 |
| 不变现 | 不好意思收钱 |
| 不圈粉 | 没有"人设锚点" |

```
人设 = 身份 + 技能 + 性格 + 记忆点
```

**牛景胜的人设应该是：**

> "一个河南农民，蓝天救援队副队长，432次出生入死救援，负债50万还在做好事，太极拳一级拳师"

这个太长了，需要简化。

### 爆款内容公式

**救援故事 = 冲突 + 悬念 + 情绪 + 结局**

```
前3秒：要么死，要么活！（冲突）
中间：紧张救援过程（悬念）
结尾：成功了/或失败了（情绪）
```

具体选题：

| 选题 | 预估播放 |
|------|----------|
| 洪水救援 | 50万+ |
| 山野搜救 | 30万+ |
| 车祸救人 | 20万+ |
| 异物卡喉 | 100万+（实用） |

这是**差异化竞争力**。
